NLP Микросервис на FastAPI

Проект представляет собой веб-сервис для обработки текстов с использованием методов NLP (Natural Language Processing). Реализован на FastAPI с полным набором NLP-методов для анализа текстов на русском языке.
Технический стек

    Backend: FastAPI, Uvicorn

    NLP библиотеки: NLTK, pymorphy3, Natasha, scikit-learn, Gensim

    Математика: NumPy

    Клиент: Tkinter (GUI), requests

Структура проекта
text

project/
├── server/           # Серверная часть
│   ├── api.py       # Эндпоинты FastAPI

Установка и запуск
1. Установка зависимостей
bash

# Создание виртуального окружения
python -m venv venv

# Активация (Linux/Mac)
source venv/bin/activate
# Активация (Windows)
# venv\Scripts\activate

# Установка библиотек
pip install -r requirements.txt

2. Запуск сервера
bash

cd project/server
python -m uvicorn api:app --reload

Сервер будет доступен по адресу: http://localhost:8000
3. Запуск клиента (в отдельном терминале)
bash

cd project/client
python client.py

Доступные методы NLP

Сервер предоставляет 9 методов анализа текста:
Математические методы

    TF-IDF - оценка важности слов в документах (реализация на NumPy)

    Bag-of-Words - векторное представление текстов (реализация на NumPy)

    LSA (Латентный семантический анализ) - понижение размерности

    Word2Vec - векторные представления слов

Лингвистические методы (NLTK + pymorphy3)

    Токенизация - разбиение текста на слова и предложения

    Стемминг - приведение слов к основе

    Лемматизация - приведение слов к нормальной форме

    POS-разметка - определение частей речи

    NER (Named Entity Recognition) - извлечение именованных сущностей

API Эндпоинты
Основные эндпоинты

    GET / - проверка работы сервера

    POST /upload-and-process - универсальный эндпоинт для загрузки файлов

Прямые эндпоинты

    POST /tf-idf - TF-IDF анализ

    POST /bag-of-words - Мешок слов

    POST /lsa - Латентный семантический анализ

    POST /word2vec - Word2Vec векторизация

Эндпоинты NLTK

    POST /text_nltk/tokenize - токенизация

    POST /text_nltk/stemming - стемминг

    POST /text_nltk/lemmatize - лемматизация

    POST /text_nltk/pos - POS-разметка

    POST /text_nltk/ner - извлечение сущностей

Использование клиента

Клиент предоставляет графический интерфейс для:

    Выбора текстового файла (.txt)

    Выбора метода анализа через меню

    Отправки файла на сервер

    Просмотра результатов

Требования к данным

    Файлы должны быть в формате .txt

    Кодировка: UTF-8

    Каждая строка файла обрабатывается как отдельный документ
